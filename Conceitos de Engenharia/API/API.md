### Conceito de APIs

##### O que é uma API?
- Permite que dois componentes de software se comuniquem usando um conjunto de definições e protocolos.

##### O que significa API?
- Application Programming Interface (Interface de Programação de Aplicação), isto é:
	1. A palavra "Aplicação" refere-se a qualquer software com uma função distinta.
	2. A palavra "Interface" ser vista como um contrato de serviço entre duas aplicações. Neste contrato, é definido como as duas aplicações vão se comunicar usando solicitações e respostas.

##### Tipos de APIs

1. APIs SOAP
2. APIs RPC
3. APIs WebSocket
4. [[APIs REST]]

#### O que é um Endpoint?
- São pontos de contato finais no sistema de comunicação da API.
- Estes incluem URLs de servidores, serviços e outros locais digitais específicos de onde as informações são enviadas e recebidas entre sistemas.
- Os endpoints da API são fundamentais por dois motivos principais:
	1. Segurança:
		- Os endpoints da API tornam o sistema vulnerável a ataques. O monitoramento da API é crucial para impedir o uso indevido.
	2. Perfomance:
		- Os endpoins da API, especialmente os de alto tráfego, podem causar gargalos e afetar a performance do sistema.
